# Семинар 5 — Макро-агент: ReAct + Tool Use

Строим ReAct-агента с пятью инструментами поверх живых сервисов ЦБ РФ и
Росстата: курс валют, ключевая ставка, инфляция, безработица, калькулятор. Агент сам решает, какие
инструменты и в каком порядке звать, чтобы ответить на макро-вопрос
настоящими числами.
