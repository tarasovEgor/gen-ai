# Семинар 6 — Мультиагент: Планировщик-Исполнитель-Критик

Оборачиваем одиночного макро-агента из С5 в мультиагентный паттерн
Планировщик-Исполнитель-Критик: Планировщик декомпозирует сложный
вопрос в план подвопросов, Исполнитель отвечает
на каждый, Критик проверяет результат и при необходимости отправляет
на доработку.
